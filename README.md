# InsightForge AI

## Problem

Businesses collect data but often struggle to convert it into actionable decisions.

InsightForge AI bridges that gap by combining:

- Data Quality Assessment
- Business Analytics
- Revenue Forecasting
- Executive Recommendations

## Version 1 Scope

- CSV Upload
- Data Validation
- Data Health Score
- KPI Dashboard
- Revenue Forecasting
- Executive Summary

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-Learn
